# Code Snapshot

This directory contains renamed inspection copies of scripts associated with the
submitted manuscript. The renamed filenames are meant to be readable to a
reviewer:

- `figure_drivers/`: submitted main and supplementary figure drivers.
- `analysis_support/`: model, grouping, distance, and calibration support code.
- `original_project_helpers/`: legacy project helpers retained for provenance.

The scripts preserve original imports and internal module names, so they are
source snapshots rather than standalone package modules. Full reruns require the
original project data layout and third-party inputs listed in
`../DATA_SOURCES.md`.
